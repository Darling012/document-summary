IPC、RPC、RMI 关系

IPC inter-process communication 进程间通信 RPC（Remote Procedure Call Protocol）远程过程调用协议 RMI （Java Remote Method Invocation） Java 远程方法调用 gRPC Google Remote Procedure Call IPC和RPC的区别_二仪式的博客-CSDN博客_ipc rpc 有两种类型的进程间通信 (IPC)。 　　本地过程调用 (LPC) LPC 用在多任务操作系统中，使得同时运行的任务能互相会话。这些任务共享内存空间使任务同步和互相发送信息。 　　远程过程调用 (RPC) RPC 类似于 LPC，只是在网上工作。RPC 开始是出现在 Sun 微系统公司和 HP 公司的运行 UNIX 操作系统的计算机中。
